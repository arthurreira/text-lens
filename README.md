# Text Lens – Hover Magnifier

Text Lens is a lightweight Chrome extension that lets you **magnify readable text under your cursor** by holding the `Control` key while hovering.

It’s designed for readability, accessibility, and quick text inspection without zooming the entire page.

---

## ✨ Features

- Hold **Control (Ctrl)** + hover to magnify text
- Works on most websites (`<all_urls>`)
- Lightweight and fast (no page reloads)
- Customizable settings (via Options page)
- Minimal UI interference
